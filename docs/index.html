Analisi regionale dei dati COVID-19 Italia

Questa pagina ha l'obiettivo di illustrare i grafici andamentali dell'emergenza con granularità regionale.

Sono utilizzati i dati ufficiali pubblicati dal Dipartimento della Protezione Civile, elaborati su base regionale. Il repository ufficiale, con il codice sorgente utilizzato per le elaborazioni, è disponibile a questo indirizzo GitHub: è un fork del repository ufficiale.

Il Dipartimento della Protezione Civile ha elaborato un cruscotto geografico interattivo raggiungibile agli indirizzi http://arcg.is/C1unv (versione desktop) e http://arcg.is/081a51 (versione mobile), contenente informazioni dettagliate sull'andamento nazionale.
